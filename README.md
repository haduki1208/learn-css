# 概要

- scss と styled-components の比較

# 使い方

## scss

```sh
$ cd learn-scss/
$ npm i
$ npm run sb
# localhost:5000にアクセス
```

## styled-components

```sh
$ cd learn-styled-components/
$ npm i
$ npm run sb
# localhost:5001にアクセス
```
